## Objective
Write a Java program using a queue to generate the first `n` binary numbers.

## Requirements
1. Use a queue data structure implemented with Java arrays to generate the first `n` binary numbers (from `1` to `n` in binary representation). A binary number represents values using only 0s and 1s.
2. Implement this queue using a Java array.
3. Write a method `generateBinaryNumbers(int n)` that returns a list of the first `n` binary numbers.
4. Use the `BinaryNumberGenerator` class in the starter code.
5. The starter code includes a set of tests to help you verify your work. To run the tests, right-click on the `BinaryNumberGeneratorTest` class in the `src/test/java` directory and select **Run 'All Tests'**.
   
### Example inputs and expected outputs
**Example 1:**  
Input: `n = 5`  
Output: `[1, 10, 11, 100, 101]`

**Example 2:**  
Input: `n = 10`  
Output: `[1, 10, 11, 100, 101, 110, 111, 1000, 1001, 1010]`

## Practical relevance
This exercise reinforces foundational concepts in data structures and real-world problem-solving with Java. You will also learn how a queue can simplify otherwise complex iterative problems like binary number generation.
