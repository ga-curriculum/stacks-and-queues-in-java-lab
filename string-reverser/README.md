## Objective
Write a Java program to reverse a string using Java’s `Stack` class.

## Requirements
- The program should take a string as input and return the reversed string using the operations provided by Java’s `Stack` class (`push`, `pop`, and `isEmpty`).
- Avoid using other string manipulation methods like `StringBuilder.reverse()`.
- Use the `StringReverser` class in the starter code.
- Create a method `reverseString` that takes a string input and returns its reversed version.
- Use `Stack` from `java.util` and its operations to reverse the string.
- The starter code includes a set of tests to help you verify your work. To run the tests, right-click on the `StringReverserTest` class in the `src/test/java` directory and select **Run 'All Tests'**.

### Example inputs and expected outputs
```plaintext
Input: "hello"
Output: "olleh"

Input: "Java"
Output: "avaJ"

Input: "Stack"
Output: "kcatS"

Input: ""
Output: "" (empty string remains empty)
```

## Practical relevance
- This exercise addresses a common problem in syntax validation, compiler design, and text editors.
- This exercise reinforces understanding of `Stack` operations like `push`, `pop`, and `isEmpty`.
